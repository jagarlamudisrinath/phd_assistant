## Update Protocol

### After Each Graded Assignment

1. **Extract feedback**:
   - Read professor comments carefully
   - Note what was praised
   - Note what was criticized
   - Identify specific recommendations

2. **Update professor section**:
   - Add to "Common feedback themes"
   - Update "Success patterns" if high grade
   - Add to "Pet peeves" if flagged as issue

3. **Look for patterns**:
   - Compare to previous feedback from same professor
   - Identify recurring themes
   - Adjust JARVIS calibration accordingly

### Quarterly Review

**End of each term**:
- Review all feedback for each professor
- Identify strongest preferences
- Update grading tendencies summary
- Refine success patterns

**Cross-professor analysis**:
- What's universal vs. individual preference
- Department or discipline-specific norms
- PhD program-wide expectations

## Initial Calibration Notes

**First few assignments** (before patterns emerge):
- Use general PhD-level academic standards
- Follow rubric criteria strictly
- Be conservative with academic tone
- Prioritize clear organization and strong evidence

**After 3-5 graded assignments per professor**:
- Patterns should become clear
- Calibrate JARVIS to professor-specific style
- Leverage successful approaches
- Avoid identified issues

**Ongoing refinement**:
- Continuously update as feedback is received
- Adapt to any changes in professor expectations
- Share insights across similar courses/professors
