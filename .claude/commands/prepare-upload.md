# Prepare Assignment for Upload

Assignment path: $ARGUMENTS

I'll help you prepare for submission by:

1. **Reading instructions.md** - Understanding all required deliverables and submission requirements
2. **Verifying formatted files exist**:
   - ✓ DOCX files with proper naming (e.g., `Jagarlamudi_Srinath_[Assignment].docx`)
   - ✓ XLSX files if required
   - ✓ Any other required formats (PDF, HTML, etc.)
3. **Running file checks**:
   - File sizes are reasonable (not empty or corrupted)
   - Filenames match submission requirements
   - All required deliverables are present
4. **Checking deadline**:
   - Extracting due date from instructions
   - Verifying you have time to submit
   - Flagging if past due or within critical window
5. **Creating upload checklist** (`UPLOAD-CHECKLIST.md`):
   - List of files to upload (with full paths)
   - Submission deadline and time zone
   - Special instructions or requirements
   - Pre-submission verification items
   - Blackboard submission steps
6. **Optional file copy**:
   - Asking if you want files copied to Desktop or Downloads
   - Making upload easier by consolidating files in accessible location

**What this command does NOT do:**
- Does not automatically upload to Blackboard (you do this manually)
- Does not modify your assignment files
- Does not submit or send anything

**After this command:**
1. Review the `UPLOAD-CHECKLIST.md` file
2. Navigate to Blackboard assignment page
3. Upload the files listed in the checklist
4. Verify upload success
5. Submit the assignment
